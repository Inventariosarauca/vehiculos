Configuración de Firebase Firestore
Configuración embebida en el archivo
La configuración Firebase está guardada directamente en el código del archivo HTML. El sistema se conectará automáticamente en cualquier dispositivo o red sin configuración adicional.
Para cambiarla: edite el bloque FB_CONFIG al inicio del script en el archivo HTML.

También puede pegar una nueva configuración abajo y hacer clic en Conectar para cambiar de proyecto Firebase en esta sesión.

Pasos para obtener tu configuración Firebase:
1. Ve a console.firebase.google.com → Crea proyecto → Habilita Firestore Database
2. Firestore → Reglas → pega: allow read, write: if true; → Publicar
3. Configuración del proyecto → Aplicaciones web → Registrar app → Copia el objeto firebaseConfig
Publicar en GitHub Pages (opcional)

Para que el sistema esté disponible desde cualquier navegador sin instalar nada:

1. Ve a github.com → Crear repositorio público → "sige-plantas-judicial"
2. Sube este archivo HTML como index.html
3. Settings → Pages → Branch: main / root → Save
4. URL disponible en: https://tu-usuario.github.io/sige-plantas-judicial/
